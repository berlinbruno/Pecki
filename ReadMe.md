# 🐦 Pecki

**Pecki** is a privacy-first Android personal finance app built with modern Android development tools. It helps users track income, expenses, budgets, and investments while keeping all data stored locally on the device.

## ✨ Features

* 💰 Income & Expense Tracking
* 📩 SMS Transaction Import
* 📊 Spending Insights
* 🎯 Budget Management
* 📈 Investment Tracking
* 🔍 Transaction Search & Filters
* 🔒 Offline & Privacy First

## 🛠 Tech Stack

* Kotlin
* Jetpack Compose
* Material Design 3
* MVVM + Clean Architecture
* Hilt
* Room
* DataStore
* Kotlin Coroutines & Flow
* GitHub Actions
